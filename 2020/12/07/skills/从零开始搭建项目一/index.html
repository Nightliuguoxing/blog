环境及软件

警告 ⚠ 一定要配置好环境变量 !!!

警告 ⚠ SpringBoot 内嵌 Tomcat 所以无需配置使用 Tomcat!

Java: JDK 1.8
MAVEN: 3.6.3
数据库: Microsoft SQL Server 2008 R2
编辑器: Intellij IDEA 2018.3.6

Java环境变量

警告 ⚠ 一定要换成自己的路径 !!!

JAVA - 下载地址

参考 - 配置Java环境变量

  • JAVA_HOME: C:\ProgramFiles\Java\jdk-8.0.101 (换成自己的路径)
  • PATH: %JAVA_HOME%\bin (固定写法禁止更改)
  • CLASS_HOME: .;%JAVA_HOME%\lib (固定写法禁止更改)

Maven 库

  • PATH: %MAVEN_HOME%\bin (固定写法禁止更改)

  • localRepository: (换成自己的路径)

  • MIRROR: (固定写法禁止更改)
