TELNET配置

路由器命令
//进特权模式的密码
R1(config) # enable password <密码>(明文)
R1(config) # enable secret <密码>(密文,级别高,覆盖明文)
R1(config) # line vty 0 4
R1(config-line) # password <密码>(远程登录的密码)
R1(config-line) # login
R1(config-line) # exit

计算机命令
telnet <ip>
